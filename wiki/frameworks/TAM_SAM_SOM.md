# Framework: TAM / SAM / SOM 분석

## 개요
프로젝트의 시장 규모를 상단(TAM)부터 1년차 목표치(SOM)까지 깔때기(Funnel) 구조로 산출한 지표입니다.

## 시장 규모 계층

| 층위 | 추정 규모 | 근거 |
|---|---|---|
| **TAM (상단)** | 약 4,100만 명 | 대한민국 만 18~79세 성인 |
| **TAM (중단)** | 약 2,000만 명 | 경제이해력 평균 60점 미달 (한국은행, 기재부 데이터) |
| **TAM (협단)** | 약 1,000만 명 | 실제 학습 의향 존재 인구 |
| **교사 TAM** | 약 20만 명 + α | 전국 초/중/고 사회 계열 교사 등 |
| **SAM (학습자)** | 약 500~700만 명 | 디지털 접근 + 학습 의향 + 체계형 선호 교집합 |
| **Strict SAM** | 약 300~450만 명 | SAM 심화분석 필터 적용 |
| **Early SAM** | 약 100~200만 명 | MVP 대상 타겟 (Q1 각성 초심자) |
| **SOM (장기)** | 약 50~100만 명 | SAM의 10~20% (10년 단위, EBS 성장 궤적 참고) |
| **SOM (1년차)** | **약 300~1,000명** | 1년차 **완주 학습자** 기준 (보수~순조 시나리오) |

## 1년차 SOM 세부 시나리오
- **보수 시나리오**: 1년차 콘텐츠 40~60편, L3 활성 3,000명, **L4 완주자 300명**, 실사용 교사 20명
- **순조 시나리오**: 1년차 콘텐츠 80~100편, L3 활성 1만 명, **L4 완주자 1,000명**, 실사용 교사 50명

## 결론 및 지향점
1년차 완주 학습자 300명은 상업적 유료 SaaS 기준으로는 초라할 수 있으나, 비영리 공공 인프라 모델([Financial Sustainability Model](../solutions/Financial_Sustainability_Model.md))에서는 후원 전환 및 공공기관 신뢰 증거로 작동하는 **충분히 건강한 출발선**입니다.
