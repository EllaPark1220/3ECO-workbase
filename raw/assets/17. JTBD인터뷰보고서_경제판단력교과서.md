### 1.1 수행 결과

- 21건 인터뷰 모두 계획서의 스크리닝 3층 필터(페르소나 매핑 → 전환 사건 확인 → 대체재 사용 경험) 통과자로 모집

### 1.2 핵심 발견 3가지

1. **Q1 각성 초심자의 Pain은 합성 프로필과 거의 일치**하되, **실제 언어는 더 정서적**이다. "점만 찍힌다", "번역기 같다", "천천히 가도 된다는 허락" 같은 표현은 합성에 없었던 언어.
2. **원칙 5(1편=1교안=1장)는 학습자/교사 간 분기된 반응**을 낳는다. Q1-B(정해민·박건우)에게는 장벽, 교사(장은혜·김수경)에게는 강점.
3. **새 Pain 3개 이상 후보 등장** — 사회적 부끄러움(NP1), AI 번역기 격하(NP2), 재개 위치 저장(NP3).

## 3. Pain 및 대체재 분석

### 3.1 AOS/DOS 18개 Pain의 실제 언어 맵

| Pain ID | Pain 요약 | 인터뷰에서 포착된 실제 언어 |
|---|---|---|
| **P1** | 체계감 부재 | "점이 찍히긴 하는데 선이 안 그어져요" (박지훈) / "AI는 번역기 같아요. 내 지식은 아니에요" (김태영) |
| **P4** | 신뢰 기준 부재 | "빠른 수익 약속하는 콘텐츠 톤이 저한테 2024년 4월로 돌아가는 방아쇠예요" (이수민) |
| **P5** | 깊이·접근성 공백 | "무겁고 얕고 중간이 없어요" (임형식) / "맨큐 50페이지에서 멈췄어요" (이수민) |
| **P7** | 판단 기준 부재 | "열 명이 열 명 다 다른 말 해요" (정성훈) |
| **P13** | 수업 준비 부담 | "재료부터 만들어야 하는 구조가 저를 지치게 해요" (김수경) |
| **P14** | 학생 자기학습 경로 부재 | "학생한테 링크 주면 알고리즘이 다른 영상 보여줘요" (박민아) |
| **P15** | 저시력 접근성 | "경제 콘텐츠는 저를 배려한다고 말한 적 없어요" (장한석) |
| **P16** | 세션 단편화 | "5분짜리가 3번 쌓이면 15분이 돼요. 그렇게 생각해요" (류지원) |

> 실제 언어가 합성 프로필 언어보다 **정서적·구체적**이라는 점이 가장 뚜렷한 차이. 마케팅 메시지와 콘텐츠 도입부 설계 시 **사용자 언어 그대로 인용**하는 것이 추천됨.

### 3.2 대체재 Hiring/Firing 사슬

| 대체재 | 주 고용 이유 (Hiring) | 주 해고 이유 (Firing) |
|---|---|---|
| **ChatGPT·AI** | 즉답·맞춤형 설명·무료 | 누적되지 않음·"내 지식 아님"·한국 세법 오류 |
| **경제 유튜브** | 알고리즘 노출·재미 | 후킹 톤·상충 정보·자극적 썸네일 |
| **경제학 책 (전문서)** | 깊이·권위 | 무거움·페이스 불일치·50페이지 벽 |
| **뉴스레터** | 짧음·매일 도달 | 아침마다 죄책감·누적 감각 부재 |
| **유료 강의** | "돈 낸 것"의 강제력 | 후킹 톤·과장 마케팅·구독 피로 |
| **FP·PB 상담** | 전문가 권위·구체성 | 상품 판매 의심·검증 언어 부재 |

**해석**: 각 대체재는 특정 순간에 고용되지만, **누적·신뢰·판단 기준이라는 공통 공백**에서 해고된다. 본 프로젝트의 포지션은 이 세 공백을 겨냥함.

## 4. 4가지 전환 동인 (Push·Pull·Habits·Anxieties) 분석

### 4.1 Push (상황의 밀어냄)

**공통 패턴**: 구체적 이벤트 + 사회적·감정적 충격

| 패턴 | 사례 |
|---|---|
| 사회적 비교 충격 | 박지훈 (친구 결혼식 대화에서 자기 무지 자각) |
| 금전 손실 경험 | 이수민 (투자 실패 -60%) / 장한석 (은행 상품 정보 비대칭 피해) |
| 의사결정 마감 압박 | 정성훈 (3년 내 주택 구매) / 박건우 (2027년 은퇴) |
| 직업적 자존감 손상 | 김수경 (학생의 "작년 데이터" 지적) |

### 4.2 Pull (새 해결책의 끌어당김)

**공통 패턴**: 본 프로젝트의 원칙 1·3·5가 Pull로 작동

| 끌림 요소 | 빈도 | 사용자 언어 |
|---|---|---|
| "선이 그어지는 감각" | 다수 Q1-A | 체계·진도·스탬프 |
| "후킹 없는 진중한 톤" | 이수민 유형 | "이거 제가 기다렸던 거예요" |
| "선택을 맡길 수 있는 구조" | 이수민·장유리 | "선택지 많으면 또 망해요" |
| "매체 선택권" | 한정숙·장한석 | "책이나 글 위주로 볼 거예요" |

### 4.3 Habits (발목 잡는 습관)

| 습관 | 영향 |
|---|---|
| "미루기" | 박지훈 — "내일 해야지" 반복 |
| "책 50페이지 벽" | 이수민·박지훈 공통 |
| "유튜브 알고리즘 수동 소비" | 다수 |
| "무료=저품질" 편견 | 조기태 유형, 박건우 일부 |

### 4.4 Anxieties (새 해결책에 대한 불안)

| 불안 | 대상 페르소나 | 본 프로젝트 대응 가능성 |
|---|---|---|
| 완주 실패 | 박지훈·이수민 | 짧은 단위 · 스탬프 맵 진도 |
| 무료=저품질 | 박건우·박지훈 | 원칙 3 · 개정 이력 명시 |
| 제작자 톤 변질 | 이수민 | 원칙 3(속도보다 신뢰) 준수 |
| 교육과정 미정합 | 김수경 | 성취기준 맵핑 문서 추가 |
| 지속 가능성 (운영자 번아웃) | 김수경 | KSF 4 (페이스 가드레일) |

## 5. 검증 명제 결과

### 5.1 명제별 판정 상세

**M1 · 체계감 부재가 이탈 주원인** ✅ 검증
- Q1-A·B 전원이 "여러 시도 후 누적 감각 부재로 이탈" 패턴 확인
- "점만 찍힘", "번역기 같음"이라는 두 가지 실제 표현으로 지지

**M2 · 진중한 톤이 지속 낳음** ✅ 검증
- 이수민 유형 전원이 후킹 톤에 대한 강한 거부감 표명
- "유료 강의도 후킹이면 해고" — 가격보다 톤이 우선

**M3 · AI가 여정에서 못 이김** ✅ 검증
- 박지훈 "점만 찍힘", 김태영 "번역기", 박건우 "한국 세법 오류"
- 대체재가 아닌 **보조 도구로 격하되는 패턴** 일관

**M4 · 교사 수업 준비 시간이 채택 변수** ✅ 검증
- 교사 3인 전원이 주당 2~4시간의 자료 제작 부담을 핵심 Pain으로 지목
- 김수경: "주당 3~4시간 돌려받는 가치"

**M5 · 매체 통합 상호 강화** ✅ 부분 검증
- 한정숙·장한석은 **글·책 위주 소비**, 박지훈·이수민은 영상 위주
- 세 매체 동시 소비는 관찰되지 않았으나, **세그먼트별 매체 선택권 필요성 확인**

**M6 · 짧은 세션이 전체에 이익** ✅ 검증
- 오세은 유형뿐 아니라 박지훈·이수민도 "출퇴근 지하철에 한 편" 선호
- NP3 "재개 위치 저장"으로 구체화

**M7 · 원칙 5가 Q1-B에 장벽** ✅ 검증 (단 분기)
- 학습자 Q1-B (정해민·박건우) : **장벽**. "지금 이 질문에 필요한 편이 먼저"
- 교사 (김수경) : **강점**. "단일 버전이 오히려 편하다"

### 5.2 신규 Pain 후보 (인터뷰 중 발견)

인터뷰 이전 AOS·DOS 18개 Pain에 없었던 3개 공통 언급 Pain:

| 코드 | Pain | 원천 | 빈도 |
|---|---|---|---|
| **NP1** | 사회적 부끄러움 ("뒤처진 느낌") | 박지훈 유형 | 다수 |
| **NP2** | AI 번역기 격하 ("내 지식 아님") | 김태영·박지훈 | 다수 |
| **NP3** | 재개 위치 저장 부재 | 오세은·서다희·이수민 | 다수 |
